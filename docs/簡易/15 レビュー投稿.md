# レビュー投稿

## 概要
スキル詳細画面から評価とコメントを投稿する。

## 対象画面
- `/skills/[id]`

## API
- `createReviewAction`
- `prisma.review.create`

## 使用テーブル
- `Review`
- `Skill`
- `User`

## 入力
| 項目 | 型 | 必須 |
|---|---|---|
| skillId | string | ○ |
| rating | number | ○ |
| comment | string | - |

## 出力
- レビュー作成結果
- 詳細画面再検証

## バリデーション
- ログイン必須
- `rating`: 1〜5
- `comment`: 最大2000文字

## 異常系
| 内容 |
|---|
| スキル不存在時はエラー |
| 入力不正時はエラー |
| 未ログイン時はログインへ誘導 |

## シーケンス
利用者 → レビュー送信 → 認証 → スキル存在確認 → 入力検証 → `Review`作成 → 詳細再表示

## 備考
- 現状、予約済みユーザーのみ投稿可などの制限は未実装
