# 提供者向け予約一覧

## 概要
自分が提供しているスキルに入った予約を、未来・過去タブで一覧表示する。

## 対象画面
- `/reservations/my`

## API
- `app/reservations/my/page.tsx` Server Component
- `prisma.reservation.findMany`
- `prisma.reservation.count`

## 使用テーブル
- `Reservation`
- `Skill`
- `User`

## 入力
| 項目 | 型 | 必須 |
|---|---|---|
| tab | string | - |
| page | number | - |

## 出力
- 予約日時
- スキル名
- 予約メッセージ
- ステータス
- 詳細リンク
- ページネーション

## バリデーション
- ログイン必須
- `tab`: `future` または `past`
- `page`: 1以上

## 異常系
| 内容 |
|---|
| 未ログイン時はログインへ誘導 |
| 該当予約なしは空表示 |

## シーケンス
提供者 → `/reservations/my` → 認証 → `Reservation`検索 → 一覧表示

## 備考
- 画面タイトルは「自分の予約」だが、実装上は提供者向け予約一覧
