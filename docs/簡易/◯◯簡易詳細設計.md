# 予約作成

## 概要
ユーザーがスキルに対して予約を作成する。

## 対象画面
- スキル詳細画面

## API
POST /api/skills/{id}/reservations

## 使用テーブル
- reservations
- skills
- users

## 入力
| 項目 | 型 | 必須 |
|---|---|---|
| skill_id | number | ○ |
| reserved_at | datetime | ○ |
| message | string | - |

## 出力
```json
{
  "message": "予約を作成しました"
}
```

## バリデーション
- 同一日時の重複予約不可
- 自分のスキルには予約不可
- 日時必須

## 異常系
| ステータス | 内容 |
|---|---|
| 401 | 未ログイン |
| 403 | 自分のスキル |
| 422 | バリデーションエラー |
