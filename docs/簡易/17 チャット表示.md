# チャット表示

## 概要
予約に紐づく会話とメッセージ一覧を表示し、メッセージ送信を行う。

## 対象画面
- `/conversations/[id]`

## API
- `app/conversations/[id]/page.tsx`
- `sendMessageAction`
- `prisma.conversation.findUnique`
- `prisma.message.create`

## 使用テーブル
- `Conversation`
- `Message`
- `Reservation`
- `Skill`
- `User`

## 入力
| 項目 | 型 | 必須 |
|---|---|---|
| id | string | ○ |
| conversationId | string | ○ |
| body | string | ○ |

## 出力
- メッセージ一覧
- 送信フォーム
- 戻るリンク

## バリデーション
- ログイン必須
- 会話ID必須
- 本文1〜1000文字

## 異常系
| 内容 |
|---|
| 会話不存在 |
| 参加者でない |
| 予約/スキル不整合は `notFound()` |
| 送信権限なしはエラー |

## シーケンス
利用者 → `/conversations/[id]` → 認証/参加者確認 → メッセージ取得 → 表示 → 送信時 `Message`作成

## 備考
- 依頼者は `/mypage`、提供者は予約詳細へ戻る設計
