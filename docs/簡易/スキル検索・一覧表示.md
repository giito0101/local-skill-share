# スキル検索・一覧表示

## 概要
登録済みスキルを新着順で一覧表示し、キーワード・カテゴリ・エリアで検索する。

## 対象画面
- `/`

## API
- `app/page.tsx` Server Component
- `prisma.skill.findMany`
- `prisma.skill.count`

## 使用テーブル
- `Skill`
- `Review`
- `User`

## 入力
| 項目 | 型 | 必須 |
|---|---|---|
| q | string | - |
| category | string | - |
| area | string | - |
| page | number | - |

## 出力
- スキル一覧
- 平均評価用レビュー情報
- ページネーション

## バリデーション
- `q`: trim後1文字以上
- `page`: 1以上の整数
- カテゴリは `SkillCategory` に含まれる値

## 異常系
| 内容 |
|---|
| 検索条件不正時は空条件扱い |
| 該当なしの場合は空一覧 |

## シーケンス
利用者 → `/` → 検索条件検証 → `Skill`検索 → 一覧表示

## 備考
- ログイン中は自分のスキルを一覧から除外
