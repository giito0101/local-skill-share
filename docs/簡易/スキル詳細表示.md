# スキル詳細表示

## 概要
スキルの詳細情報、提供者情報、レビュー一覧、平均評価、予約導線を表示する。

## 対象画面
- `/skills/[id]`

## API
- `app/skills/[id]/page.tsx` Server Component
- `prisma.skill.findUnique`

## 使用テーブル
- `Skill`
- `User`
- `Review`

## 入力
| 項目 | 型 | 必須 |
|---|---|---|
| id | string | ○ |

## 出力
- スキル詳細
- 画像
- 料金
- エリア
- 平均評価
- レビュー一覧
- 予約ボタン

## バリデーション
- `id` が空でない文字列

## 異常系
| 内容 |
|---|
| スキル不存在時は `/` へリダイレクト |
| 自分のスキル閲覧時は `notFound()` |

## シーケンス
利用者 → `/skills/[id]` → `Skill`取得 → 権限確認 → 詳細表示

## 備考
